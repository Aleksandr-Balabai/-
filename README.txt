1 В данном проекте при помощи Vagrant поднимается 6 виртуальных машин в VirtualBox. Средствами Ansible эти машины объединяются в docker swarm cluster, где впоследствии разворачивается сервис nginx. Также Ansible проверяет доступность сервиса.

2 Файл Vagrantfile описывает количество ВМ и их конфигурацию, присваивает имена и ip адреса ВМ, открывает порты необходимые для работы docker swarm, также файл содержит скрипт установки Docker на ВМ и скрипт копирования публичного ключа на виртуальные машины для дальнейшего доступа к ним по ssh.

3 Файл main.yml содержит список задач Ansible для достижения состояния описанного в п.1

4 Файл get_out_of_cluster.yml содержит список задач Ansible для выхода машин из docker swarm cluster.

5 Доступность сервиса проверяется GET-запросом при помощи утилиты curl

6 Отказоустойчивость сервиса гарантируется самим Docker и достигается за счет декларативных комманд, конкретно в данном проекте --replicas

7 Файл Step by step guide.txt содержит пошаговые действия для запуска и остановки проекта.
